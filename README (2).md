# 🎬 Movie Recommendation System

A content-based movie recommendation system built using Python and Machine Learning that suggests similar movies based on user preferences.

---

## 🚀 Live Demo & Repository

- 🔗 Live Demo: https://lnkd.in/g7VEvTix  
- 💻 GitHub Repository: https://lnkd.in/g8ah3TBy  

---

## 🚀 Features

- Recommend movies based on similarity
- Content-based filtering using cosine similarity
- Interactive web interface using Streamlit
- Real-time recommendations
- Deployed on cloud for public access

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- Cosine Similarity  

---

## 📂 Project Structure

Movie-Recommendation-System/
│── app.py

│── README.md

---

## ⚙️ How It Works

1. Load movie dataset  
2. Perform data preprocessing and feature extraction  
3. Compute similarity scores using cosine similarity  
4. User selects a movie  
5. System recommends similar movies

## ▶️ Run Locally

git clone https://lnkd.in/g8ah3TBy
